### Ads Finder with NLP

App allows user to enter to a blog url to find the related ads and keywords.

	-   Shows keywords
	-   Shows related ads with images and URL
	
Modules Used:

	- NLTK
	- django
	- pandas
	- request
        - beautifulSoup
